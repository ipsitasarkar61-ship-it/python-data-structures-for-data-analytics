## Problem
Given two strings, determine whether one string is an anagram of another.

## Approach
- Checked if both strings have equal length
- Used a HashMap (dictionary) to count character frequency
- Reduced count using the second string
- Returned false if mismatch occurred

## Complexity
- Time Complexity: O(n)
- Space Complexity: O(n)

## Learning
